## Reconocimientos

Este proyecto utiliza las siguientes librerías y herramientas de código abierto:

- [boto3](https://github.com/boto/boto3) - AWS SDK para Python
- [botocore](https://github.com/boto/botocore) - Biblioteca base de boto3
- Y todas las dependencias listadas en `requirements.txt`

Gracias a todos los mantenedores y contribuidores de estas librerías.
